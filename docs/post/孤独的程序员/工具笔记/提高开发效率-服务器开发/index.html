服务器开发时的环境配置和开发工具

环境配置

SSH免密登录

  1. 本地.ssh/config中配置相关账号端口信息
  2. 将公钥写到远程主机：ssh-copy-id username@ip

Python环境

  1. conda创建不同python环境
  2. 安装torch以及transformers等包，推荐使用可复用脚本离线统一配置

注意区分conda和cuda：

  • conda是python环境管理的包
  • cuda是基于C++调用底层驱动的包
