SSH免密登录#

  1. 本地.ssh/config中配置相关账号端口信息
  2. 将公钥写到远程主机：ssh-copy-id username@ip

GPU环境配置#

  1. conda创建环境：conda create -n py36 python=3.6
  2. 安装torch以及transformers等包，推荐使用可复用的install.sh脚本离线统一配置
