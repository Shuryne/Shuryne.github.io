Tmux笔记#

介绍#

  • 推荐博客：Tmux使用手册
  • 优势：对比nohup，tmux有其独到的优势，如能够同时开多个后台运行程序等

使用#

  1. tmux new -s demo：新建一个名称为demo的会话
  2. tmux kill-session -t demo：关闭demo会话
  3. tmux a -t demo：进入到名称为demo的会话
  4. Ctrl + b + d：断开当前会话，会话在后台运行
  5. tmux内使用滚动：Ctrl + b，键入set -g mouse on
  6. tmux内使用选中复制：按住alt再选中即可复制选中内容
